## Installation

Using `yarn`:

```
yarn add mylibrary
```

Using `npm`:

```
npm install mylibrary --save
```

## Development

### Publish A New Release

```
yarn release patch
```

```
yarn release minor
```

```
yarn release major
```

After a new release has been created by pushing the tag, it must be published
via the GitHub UI. This triggers the final publication to npm.
